# Simulation Transient Pulse

Demonstrates transient analysis with a pulse source.

## Circuit Summary

A pulsed input drives a simple network to observe time-domain response.

## Expected Behavior

- Waveforms show rise/fall behavior and time constants.

## How To Run

1. Open `Simulation-Transient-pulse.kicad_sch` in KiCad.
2. Click `Simulate` and run the configured analyses.
3. Add probes to the key nodes to view the expected behavior.

## Notes

- Adjust pulse width and period to explore settling behavior.
