# Simulation AC Sweep

Demonstrates how to configure and run an AC sweep analysis.

## Circuit Summary

A small-signal AC source drives a simple network for frequency response analysis.

## Expected Behavior

- Magnitude and phase plots appear across the configured frequency range.

## How To Run

1. Open `Simulation-AC-sweep.kicad_sch` in KiCad.
2. Click `Simulate` and run the configured analyses.
3. Add probes to the key nodes to view the expected behavior.

## Notes

- Try logarithmic vs. linear sweeps for different views.
