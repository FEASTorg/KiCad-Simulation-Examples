# Rectifier 4d Op-Amp NPN

Demonstrates an op-amp assisted rectifier with an NPN pass device.

## Circuit Summary

The op-amp compensates diode drops to improve low-voltage rectification.

## Expected Behavior

- Output follows the input more closely near zero crossings.

## How To Run

1. Open `Rectifier-4D-OPamp-NPN.kicad_sch` in KiCad.
2. Click `Simulate` and run the configured analyses.
3. Add probes to the key nodes to view the expected behavior.

## Notes

- Compare to a standard bridge to see reduced dead zone.
