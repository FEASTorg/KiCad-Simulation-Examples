# Simulation DC Sweep

Demonstrates DC sweep analysis.

## Circuit Summary

A source is swept across a range while the output is monitored.

## Expected Behavior

- Output curves show how the circuit responds to the swept parameter.

## How To Run

1. Open `Simulation-DC-sweep.kicad_sch` in KiCad.
2. Click `Simulate` and run the configured analyses.
3. Add probes to the key nodes to view the expected behavior.

## Notes

- Use smaller steps for smoother curves.
